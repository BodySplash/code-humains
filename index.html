TDD

  1. Interdiction d'écrire de code de production, à moins de faire passer un test au vert
  2. Interdiction d'écrire plus de code de test qu'il ne faut pour qu'il ne soit rouge. La compilation que ne passe pas est un test rouge
  3. Interdiction d'écrire plus de code de production qu'il ne faut pour faire passer un test qui était rouge, au vert

Refactoring

Changer la structure interne sans changer le comportement externe.
